Aqui você encontrará dashboards interativos focados em análise de dados, visualização e geração de insights para tomada de decisão.

## Sobre este repositório
Este repositório reúne projetos práticos que demonstram minhas habilidades em:
-Análise exploratória de dados (EDA)
-Criação de dashboards interativos
-Transformação de dados com Power Query
-Modelagem de dados e DAX
-Storytelling com dados

Os projetos foram desenvolvidos com foco em aplicações reais de negócio.
